## Getting Started

if you want this project, follow the next intruccions:
To get a local copy up and running follow these simple example steps.

install the next npm items to run i:
-  `npm install`
-  `npm install webpack webpack-cli --save-dev`
- run  `npm run start`
- enjoy :smile:
